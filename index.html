In Loving Memory of WoC

2020 - 2025

WoC was a highly active and vibrant programming Discord server, boasting a community of approximately 79,121 members.

Tragically, the server was compromised when its owner, Nick, reportedly fell victim to a robux scam, leading to the unfortunate loss of the community's digital home.

Discord if you don't recover the server I will be sad so will the rest of the survivors.
